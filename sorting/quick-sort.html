Pseudo Code

procedure quickSort(arr, low, high):
    if low < high:
        pivot = partition(arr, low, high)
        quickSort(arr, low, pivot - 1)
        quickSort(arr, pivot + 1, high)

procedure partition(arr, low, high):
    pivot = arr[high]
    i = low - 1
    for j = low to high - 1:
        if arr[j] <= pivot:
            i = i + 1
            swap(arr[i], arr[j])
    swap(arr[i + 1], arr[high])
    return i + 1

Algorithm Details

Time Complexity

- Best Case: O(n log n)
- Average Case: O(n log n)
- Worst Case: O(nÂ²) when array is already sorted or reverse sorted

Space Complexity

O(log n) - Due to recursive call stack

How it Works

Quick Sort works by selecting a 'pivot' element and partitioning the array around it, such that smaller elements are moved to the left of the pivot and larger elements to the right. This process is then recursively applied to the subarrays. The efficiency of Quick Sort heavily depends on the choice of the pivot element.
