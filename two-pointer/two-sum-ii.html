Pseudo Code

function twoSum(numbers, target):
    left = 0
    right = numbers.length - 1
    
    while left < right:
        currentSum = numbers[left] + numbers[right]
        
        if currentSum === target:
            return [left + 1, right + 1]  // 1-indexed
        else if currentSum < target:
            left++  // need a bigger sum
        else:
            right--  // need a smaller sum
    
    return []  // no solution found

Algorithm Details

Time Complexity

O(n) where n is the length of the array

Space Complexity

O(1) - constant extra space

How it Works

The Two Sum II algorithm uses a two-pointer approach to find two numbers that add up to a specific target:

1. We use two pointers: one starting from the left (at index 0) and one from the right (at the last index)
2. At each step, we calculate the sum of the elements at the left and right pointers
3. If the sum equals the target, we've found our answer
4. If the sum is less than the target, we move the left pointer right to increase the sum
5. If the sum is greater than the target, we move the right pointer left to decrease the sum

This approach works because the array is sorted. We can systematically eliminate possibilities and find the solution in linear time.
