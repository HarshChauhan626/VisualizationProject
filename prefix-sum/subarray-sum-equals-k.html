Pseudo Code

function subarraySum(nums, k):
    count = 0
    prefix_sum = 0
    sum_counts = {0: 1}
    
    for num in nums:
        prefix_sum += num
        
        # If (prefix_sum - k) exists in sum_counts,
        # it means we found a subarray that sums to k
        if (prefix_sum - k) in sum_counts:
            count += sum_counts[prefix_sum - k]
        
        # Update the count for current prefix sum
        sum_counts[prefix_sum] = sum_counts.get(prefix_sum, 0) + 1
    
    return count

Algorithm Details

Problem Description

Given an array of integers and an integer k, find the total number of continuous subarrays whose sum equals to k.

Time Complexity

O(n) where n is the length of the array

Space Complexity

O(n) for storing prefix sums

How it Works

The algorithm uses prefix sums and a hash map to efficiently find subarrays that sum to k:
1. Calculate prefix sums as we iterate through the array
2. For each prefix sum, check if (prefix_sum - k) exists in the hash map
3. If it exists, it means we found a subarray that sums to k
4. Update the hash map with the current prefix sum
