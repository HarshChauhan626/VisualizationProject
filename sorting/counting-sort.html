Pseudo Code

procedure countingSort(arr):
    n = arr.length
    max = findMax(arr)
    // Create counting array
    count = new array[max + 1]
    // Initialize count array
    for i = 0 to max:
        count[i] = 0
    // Store count of each element
    for i = 0 to n-1:
        count[arr[i]]++
    // Calculate cumulative count
    for i = 1 to max:
        count[i] += count[i-1]
    // Build output array
    output = new array[n]
    for i = n-1 to 0:
        output[count[arr[i]]-1] = arr[i]
        count[arr[i]]--
    // Copy output array to original array
    for i = 0 to n-1:
        arr[i] = output[i]

Algorithm Details

Time Complexity

- Best Case: O(n + k)
- Average Case: O(n + k)
- Worst Case: O(n + k)
where k is the range of input

Space Complexity

O(k) - Additional space for counting array

How it Works

Counting Sort is a non-comparative sorting algorithm that works by counting the number of occurrences of each element in the array. It then uses this information to place each element in its correct sorted position. The algorithm is particularly efficient when sorting integers with a known range, as its time complexity is linear in both the number of items and the range of input values.
