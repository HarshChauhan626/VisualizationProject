Pseudo Code

function minPathSum(grid):
    m, n = len(grid), len(grid[0])
    // Initialize dp array
    dp = [[0] * n for _ in range(m)]
    // Base case: first cell
    dp[0][0] = grid[0][0]
    // Fill first row and column
    for i = 1 to m-1: dp[i][0] = dp[i-1][0] + grid[i][0]
    for j = 1 to n-1: dp[0][j] = dp[0][j-1] + grid[0][j]
    // Fill dp array
    for i = 1 to m-1:
        for j = 1 to n-1:
            dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]
    return dp[m-1][n-1]

Algorithm Details

Time Complexity

O(m*n) where m and n are the grid dimensions

Space Complexity

O(m*n) for the dp array, can be optimized to O(n)

How it Works

The algorithm works by:
1. Using dynamic programming to store minimum path sum to each cell
2. Base case: first cell value
3. Fill first row and column with cumulative sums
4. For each cell (i,j):
   - MinSum[i][j] = min(MinSum[i-1][j], MinSum[i][j-1]) + grid[i][j]
   - Choose minimum path from above or left
5. Return the minimum path sum to the bottom-right cell
