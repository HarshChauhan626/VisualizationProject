Find All Missing Numbers - Cyclic Sort

Problem Statement

Given an array nums of n integers where nums[i] is in the range [1, n], return an array of all the integers in the range [1, n] that do not appear in nums.

Example:
Input: nums = [4,3,2,7,8,2,3,1]
Output: [5,6]
Explanation: The numbers 5 and 6 are missing from the array.

Difficulty: Easy-Medium
Pattern: Cyclic Sort
Follow up: Could you do it without extra space and in O(n) runtime?

Pseudo Code

function findDisappearedNumbers(nums):
    i = 0
    // Phase 1: Cyclic sort
    while i < nums.length:
        j = nums[i] - 1  // correct index for current element
        if nums[i] != nums[j]:
            swap(nums, i, j)  // place element at correct position
        else:
            i++  // move to next element
    
    // Phase 2: Find missing numbers
    missingNumbers = []
    for i = 0 to nums.length - 1:
        if nums[i] != i + 1:
            missingNumbers.add(i + 1)
    
    return missingNumbers

Algorithm Details

Time Complexity

O(n) where n is the length of the array

Space Complexity

O(1) - only constant extra space (excluding output array)

How It Works

1. Phase 1 - Cyclic Sort: Place each number at its correct position (number n goes to index n-1)
2. If current number equals number at its target position, move to next index
3. Otherwise, swap current number with the number at its correct position
4. Phase 2 - Find Missing: Check each position to see if it contains the correct number
5. If position i doesn't contain (i+1), then (i+1) is missing
